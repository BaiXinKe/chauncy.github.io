在该种情况下，做如下两步推导：

  1. 将expr中的引用属性移除。
  2. 进行型别推导，即使用expr和ParamType来执行型别匹配，确定T的类型
expr int const int const int&

ParamType int& const int& const int&

指针和上述情况类似，上述情况符合传统意义上的C++模板型别推导规则。

ParamType为万能引用

ParamType为T&&

此时遵照下述两个条件：

  1. 若expr为左值，则T和ParamType都将被推断为expr类型的左值引用，这是型别推导过程中唯一可以使得T推导结果为引用的情况（引用折叠）。
  2. 若expr为右值，则T和ParamType则与常规规则相同。
expr int const int const int& 27
T int& const int& const int& int
ParamType int& const int& const int& int&&

当遇到万能引用时，型别推导规则会区分实参的左右值。

  1. 移除引用属性。
  2. 移除const属性以及volatile属性，然后根据类型进行推导。

ST-GDN

前期准备

传统上计算的是交通流量（volume，每个区域的流入和流出）。

该篇文章在三个方面做出了改变：

​ 捕获的时间模式通常彼此互补，因此需要多分辨率的，即多个不同周期的交通数据采样来对数据进行建模来增强鲁棒性。

​ 递归神经网络在高阶多维时间范围内表现并不是很好（几乎不行）。

​ 地理属性相同的区域在交通模式上有着相似的表现，即使其并非临近。

为解决上述问题，提出了ST-GDN（时空图扩散网络），着眼点在特定区域交通流。

​ 多尺度自注意力网络探究多细粒度时间动态性（不同采样频率的时间相关性），使用一个聚合层来对跨多级时间动态的基础依赖进行建模。

​ 使用图注意力扩散范式开发的层次图神经网络使ST-GDN能够以联合的方式将空间语义从局部级别的空间相邻关系合并到整个城市的全局级别的交通模式中。

定义1 空间区域， 栅格化建模

定义2 交通流张量， 区域流出量和区域流入量

任务 输入区域流出量和区域流入量，输出将来时段的交通流量。

方法

时间层次模型

​ 多尺度自注意力网络来连结映射多级时间信号，抓取模式。

​ 时间分辨率

​ 在对交通数据进行多分辨率采样后，使用一种自注意力网络来从时间维度对流量变化模式进行编码。编码器建立在具有三个（query, key, value）变换矩阵的可缩放点击注意力架构上。

$
\left[
\begin{array}{l}
Q \ K \ V
\end{array}
\right]
=
E^p
\left[
\begin{array}{l}
W_Q \ W_K \ W_V
\end{array}
\right]
;
Y^p = \sigma(\frac {QK^T} {\sqrt{d}})
$

全局信息交通依赖学习

​ 根据动态流量转换模式来利用不同区域之间的全局级别的依赖关系。

​ 全局图G=(R,E)。R区域集合，E两区域之间的关系。使用注意力机制的聚合表示方法如下：

$
m^p{(i,j)\leftarrow(i’,j’)} = {||}{h=1}^{H} w_{(i,j);(i’j’)}·Y^p·W^p
$

等式右侧为特征信息沿箭头方向传播，此处赋予跨区域相关编码多头，来从不同的子区域中捕捉区域相关的语义，最后那个$W^p$矩阵为参数化投影矩阵。

注意力相关性表示方法：
$
w^h{(i,j);(i’,j’)} = \frac {exp(LR(\alpha^T[\hat{y}^p{i,j}||\hat{y{i’,j’}}]))} {\sum{(i’,j’) \in \mathcal{N}}exp(LR(\alpha^T[\hat{y}^p{i,j}||\hat{y}^p{i’,j’}]))}
$
在y中使用关联运算。
